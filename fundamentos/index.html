¿Qué es Sakila?


Sakila es una base de datos de muestra desarrollada por Mike Hillyer, ex miembro de la documentación de MySQL AB,proporciona un esquema estándar que puede usarse para ejemplos en libros, tutoriales, artículos, muestras, etc. La base de datos también permite ver características de MySQL como vistas, Procedimientos almacenados y Disparadores.
  • Mas información en: https://dev.mysql.com/doc/sakila/en/

Tablas


La base de datos Sakila cuenta con diferentes tablas.

  • Clientes
  • Películas
  • Gestiones
  • Vistas
